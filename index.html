BK's Stock Predictor

This project is a web-based Stock Price Predictor built with Python, Streamlit, and machine learning. The app allows users to input a stock symbol, view historical price data, analyze moving averages, and predict future stock prices using a Random Forest Regressor. Visualizations include comparisons of original and predicted prices, along with interactive charts for various moving averages (100, 200, and 250 days). Key Features: Dynamic stock data retrieval via Yahoo Finance API. Real-time data visualization with Matplotlib. Predictive modeling using scaled data and machine learning.
